La aplicación está desarrollada en React Native e integra NativeWind para aplicar 
estilos de manera rápida y coherente, aprovechando la sintaxis de Tailwind adaptada
a entornos móviles. Se trata de una app de música inspirada en el estilo de Spotify,
que permite explorar, buscar y reproducir canciones en una interfaz moderna y fluida.
Cuenta con un reproductor de música integrado capaz de controlar la reproducción en 
segundo plano y mostrar los controles en pantalla. Para garantizar la seguridad y 
consistencia de los datos del usuario, se usa Zod como sistema de validación, 
especialmente en formularios de registro e inicio de sesión, asegurando que las 
entradas cumplan con el formato previsto antes de enviarse al backend. El resultado 
es una experiencia completa, visualmente cuidada y con una arquitectura robusta,
ideal para móviles Android y iOS.
